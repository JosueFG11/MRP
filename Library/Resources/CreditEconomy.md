# Clarity Credit Economy / Créditos de Claridad

## Definition
- En: Consumable credits for Mr.P sessions (avg 700 per session), 3-month rollover.
- Es: Créditos consumibles para sesiones Mr.P (prom. 700/sesión), rollover 3 meses.

## Key Elements
- En: Grants: 5,000 free (Awakening); 22,000/mo Reflective; +24,000 Add-On.
- Es: Otorgamiento: 5,000 gratis (Awakening); 22,000/mes Reflective; +24,000 Add-On.
- En: Top-ups: $0.10 per 1,000 credits via Stripe.
- Es: Recargas: $0.10 por 1,000 créditos vía Stripe.

## Metrics & Implementation
- Instant balance update post-payment (<5s)  
- Balance sync <1s on Dashboard  
- Goal: Sustainable markup for 1M MAU
